# Lintament VS Code Extension

An LLM-powered linter that replaces bad code with ancient wisdom. Because sometimes, the best fix is philosophical acceptance.

## Features

- **AI-Powered Judgment**: Analyzes your code for spiritual weakness and lack of discipline
- **Ancient Wisdom**: Replace technical debt with philosophical debt
- **Zero Learning Curve**: If you can read, you can be enlightened
- **Instant Enlightenment**: Bad code is simply removed from existence

## Commands

- **Judge Current File**: Analyze and fix the currently open file
- **Judge Entire Workspace**: Analyze all files in your workspace
- **Consult the Ancients**: Get wisdom about selected code
- **Regress**: Undo the last fix (cowardice mode)

## Configuration

Open VS Code settings and search for "Lintament" to configure:

- `lintament.wisdomLevel`: Level of judgment (`gentle`, `firm`, `maximum`, `ruthless`)
- `lintament.mercyMode`: Set to true if you are emotionally fragile
- `lintament.quoteSource`: Source of wisdom (`sun-tzu`, `marcus-aurelius`, `confucius`, `your-disappointed-father`)
- `lintament.judgmentThreshold`: How bad code must be before replacement (0-1)
- `lintament.preserveComments`: Whether to preserve comments (default: false)
- `lintament.apiKey`: OpenAI API key for LLM analysis (optional, uses heuristics if not provided)
- `lintament.model`: OpenAI model to use (default: `gpt-4`)

## Installation

1. Clone this repository
2. Run `npm install` in the `extension` folder
3. Press `F5` to open a new VS Code window with the extension loaded
4. Or package the extension: `vsce package` and install the `.vsix` file

## Usage

1. Open a code file
2. Right-click and select "Lintament: Judge Current File" or use the command palette
3. Watch as your code is analyzed and potentially replaced with wisdom
